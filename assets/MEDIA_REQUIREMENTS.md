## 🔧 RECOMMANDATIONS TECHNIQUES

### Formats Acceptés
- **Images**: JPG, PNG, WebP
- **Vidéos**: MP4 (H.264), WebM

### Optimisation Images
```
- Réduire la taille sans perte de qualité
- Utiliser Tinypng.com ou similar
- Poids cible: 80-250 KB par image
- Ratio d'aspect doit être respecté exactement
```

### Optimisation Vidéos
```
- Compresser avec Handbrake ou similar
- Bitrate recommandé: 2-4 Mbps
- Résolution minimale: 720p
- Poids max: 15-20 MB
```

### Nommage Fichiers
```
Format: nom-descriptif-lowercase.extension
Exemples:
✓ hero-placeholder.jpg
✓ atelier-crochet.jpg
✓ gallery-5.jpg
✗ Hero Placeholder.jpg
✗ IMG_12345.jpg
```

### SEO & Alt Text
Chaque image doit avoir un `alt` descriptif:
```html
✓ <img src="..." alt="Mains créant au crochet, atelier attimo.">
✗ <img src="..." alt="image">
```
