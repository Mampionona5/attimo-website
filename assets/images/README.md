## 💡 Conseils

**Où trouver les images?**
- Facebook attimo. (photos existantes)
- Prise de photos lors des ateliers
- Microstock gratuit: Unsplash, Pexels (si besoin)

**Qualité photos**:
- Lumière naturelle idéale
- Mains visibles quand possible
- Couleurs: harmoniser avec la palette (#A8B89A, #F5F0E8)
- Pas de watermark visible

**Optimisation avant upload**:
1. Redimensionner à la taille exacte
2. Compresser avec Tinypng.com
3. Renommer correctement
4. Double-check les dimensions
