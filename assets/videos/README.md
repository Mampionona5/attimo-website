## 🔧 Compression Vidéo

**Outil gratuit**: Handbrake (https://handbrake.fr/)

**Paramètres recommandés**:
```
Codec: H.264
Résolution: 1280 × 720 (HD) ou 1920 × 1080 (Full HD)
Bitrate: 2-4 Mbps
Frame rate: 30 FPS
Audio: AAC, 128 kbps
```
