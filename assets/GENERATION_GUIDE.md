## ⚙️ Workflow Recommandé

### **Pour CHAQUE image:**

1. **Copier le prompt** de la section ci-dessus
2. **L'adapter si besoin** (ajouter détails personnels)
3. **Générer avec Nano Banana** (ou autre IA)
4. **Évaluer** (dimensions ok? couleurs ok? style ok?)
5. **Si mauvais:** Ajouter des modifications
   - "Add more visible hands"
   - "Make it lighter/warmer"
   - "Change background to..."
6. **Si bon:** Renommer et placer dans `assets/images/`
7. **Tester sur le site** (ouvrir dans navigateur, vérifier rendu)
8. **Passer à la suivante**

---

## 🔄 Itération (si pas satisfait)

**Techniques pour améliorer:**

❌ Mauvais résultat → Trop synthétique
```
✅ Ajouter: "Real hands, authentic, not AI-looking"
✅ Ajouter: "Documentary photography style"
✅ Ajouter: "Candid, unposed, natural"
```

❌ Mauvaises couleurs
```
✅ Ajouter: "Must include sage green and warm cream tones"
✅ Ajouter: "Color palette: #A8B89A, #F5F0E8, #C4A882"
```

❌ Pas assez de détails
```
✅ Ajouter: "Close-up shot, visible texture, sharp focus"
✅ Ajouter: "Macro photography, fine details visible"
```

❌ Mauvaises dimensions
```
✅ Relancer en spécifiant: "Aspect ratio: 16:9" ou "1:1" ou "3:2"
```

## 💾 Après Génération

### Étapes pour chaque image:

```
1. Télécharger image depuis Nano Banana
2. Vérifier dimensions exactes (avec Windows/properties)
3. Compresser si > 250KB (Tinypng.com)
4. Renommer EXACTEMENT comme dans le prompt
5. Placer dans: assets/images/[NOM_EXACT].jpg
6. Tester dans navigateur (F5 pour refresh)
7. Vérifier alignement/affichage sur mobile aussi
```
